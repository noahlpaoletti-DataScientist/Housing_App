# Data Sources

## MVP source strategy

This repository intentionally uses mock data so the product works without API keys or licensing dependencies on day one.

Current seeded sources:

- Mock Property Registry
- Mock Comparable Sales
- Mock Market Trends
- Mock Material Cost Assumptions

## Designed future connectors

- County parcel APIs
- Local assessor data
- Census API context
- FHFA House Price Index data
- Building permit datasets
- Licensed MLS feeds
- Licensed listing datasets
- Rental estimate datasets
- Construction-cost feeds
- Insurance replacement-cost datasets
- User-uploaded CSV datasets

## Legal note

The product should not scrape Zillow, Redfin, Realtor.com, MLS systems, or other proprietary listing datasets without proper licensing.
