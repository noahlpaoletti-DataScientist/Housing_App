# Product Strategy

## Positioning

The MVP is a transparent alternative to black-box home value estimates. Instead of returning a single number with little context, the platform shows the signals, weights, assumptions, and confidence behind the estimate.

## Primary users

- Brokerages and agents that need pricing support they can explain to clients.
- Investors evaluating flips, rentals, and renovation upside.
- Lenders and insurers that need value context and replacement-cost framing.
- Homeowners comparing sale timing and renovation ROI.
- Builders and contractors using scope-based cost assumptions.

## Core promise

For every property lookup, the user should understand:

- What data was used
- How much each data source influenced the result
- What assumptions were made
- Where uncertainty exists
- How market conditions and renovations affect the number

## MVP wedge

The immediate wedge is a report-first product:

- Address lookup
- Explainable valuation
- Comparable-sale logic
- Renovation ROI
- Investor calculator
- Source provenance

That combination makes the product useful even before live data integrations.
